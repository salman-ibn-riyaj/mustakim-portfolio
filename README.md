## 🛠️ Built With

This portfolio is built leveraging modern web standards and cutting-edge frontend tools:

- **Next.js** – React Framework for production-grade builds and optimized performance (App Router architecture).
- **React.js** – Component-based UI rendering.
- **Tailwind CSS** – Utility-first CSS framework for rapid and precise responsive design.
- **Framer Motion** – Production-ready motion library for fluid and smooth animations.
